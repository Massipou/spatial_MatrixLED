## spatial_MatrixLED
Spatial web interface led project 

This project is a web interface for managing stripe LED animations.

## Concept:
make stripe LED animations with a raspberry and a webserver.
Raspberry listen for the webserver via network for getting colors, changing animations, add effects ...
Webserver use audio input like microphone for generate RGB color and send it to the raspberry.
